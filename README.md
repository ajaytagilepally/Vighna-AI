# Vighna AI

🤖 A comprehensive AI application combining ChatGPT-like conversational intelligence with GitHub Copilot-like code generation capabilities.

## 🌟 Features

- **Conversational AI**: Chat interface powered by OpenAI GPT or Anthropic Claude
- **Code Generation**: AI-powered code completion and generation
- **Code Analysis**: Analyze, explain, and refactor code snippets
- **Context Awareness**: Maintain conversation history and multi-turn conversations
- **Real-time Streaming**: Stream responses for better UX
- **Multiple AI Providers**: Support for OpenAI, Anthropic Claude, and GitHub Copilot APIs
- **Code Editor**: Integrated editor with syntax highlighting
- **Conversation History**: Save and manage chat histories

## 🛠 Tech Stack

### Backend
- **Python**: FastAPI with async support for AI model integration
- **Node.js**: Express.js server with WebSocket for real-time features
- **Database**: PostgreSQL with SQLAlchemy ORM
- **Authentication**: JWT-based authentication
- **Caching**: Redis for session management

### Frontend
- **Angular**: 16+ with TypeScript
- **Angular Material**: Professional UI components
- **RxJS**: Reactive programming patterns
- **Monaco Editor**: Advanced code editor
- **Tailwind CSS**: Utility-first CSS framework
